Projects

Here are some of my projects! Most of them are focused around data science and python

Stock Market Algorithm

Used Jupyter Notebooks, sklearn, and other python libraries to analyze smp500 data. Predicted the day to day movement of stocks with r=0.99 using ML techniques such as linear regression, elastic net regularization, polynomial feature expansion, and more

Husky NightRide - UW Dubhacks

Used Google Cloud, Twilio API to automate and streamline the nightride service for UW students. Implemented multi language accessibility with English, Spanish, Mandarin and Hindi.

Algae Biofuels

Used Jupyter Notebooks, pandas and other python libraries to try to find correlations between environmental factors and lipid production in algae biofuels. Developed a website for further analysis and conclusions of results.
